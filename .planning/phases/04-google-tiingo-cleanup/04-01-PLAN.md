**Phase:** 4 — Google & Tiingo Cleanup

## Goal

Remove all Google Sheets code, dead Tiingo config, and orphaned dependencies from the codebase. Fix the missing `GROWTH_CAP` application on the Finnhub growth path (BRK-B uncapped growth bug). After this plan, the screener has no vestigial output code and the GitHub Actions workflow is clean.

---

1. `push_to_gsheets()` and its private helpers (`_apply_color_coding`, `_write_docs_tab`, `_write_markdown_tab`, `_col_letter`) no longer exist in `stock_screener.py`
2. `gspread` and `google-auth` are absent from `requirements.txt`; `screener.yml` has no `GSHEET_*` or `TIINGO_API_KEYS` environment variable references
3. Dead Tiingo config (`TIINGO_API_KEYS`, `TIINGO_DELAY_SEC`, `time.sleep(TIINGO_DELAY_SEC)`) is removed; `import time` is removed
4. `GROWTH_CAP` is applied to the Finnhub `growth_pct` path in `process_ticker()` — no path produces uncapped growth
5. `python -m py_compile stock_screener.py` exits 0 with no errors

### Task 6 — Fix missing `GROWTH_CAP` on Finnhub growth path in `process_ticker()`

**File:** `stock_screener.py`

**Context:** `compute_growth_5yr_cagr()` caps `g` at `GROWTH_CAP = 25.0`, but the Finnhub `growth_pct` path (direct from `fund["growth_pct"]`) skips this cap. BRK-B's Finnhub `epsGrowth5Y` returns ~1179% (Class A EPS basis), flowing uncapped into Lynch scoring.

**Current code** (around line 594):
```python
    # ── Growth — use Finnhub 5Y CAGR, fall back to computed CAGR ────
    g = fund["growth_pct"]
    if g is None:
        # Fallback: compute from yfinance EPS history
        g = compute_growth_5yr_cagr(fund["annual_eps"])
    if g is None:
        log.info(f"{ticker}: growth not computable, skipping valuation")
        row["Error"] = "Growth N/A"
        return row
    if g <= 0:
```

**Change:** Add a cap line after the `if g is None: return row` early exit:
```python
    # ── Growth — use Finnhub 5Y CAGR, fall back to computed CAGR ────
    g = fund["growth_pct"]
    if g is None:
        # Fallback: compute from yfinance EPS history
        g = compute_growth_5yr_cagr(fund["annual_eps"])
    if g is None:
        log.info(f"{ticker}: growth not computable, skipping valuation")
        row["Error"] = "Growth N/A"
        return row
    g = min(g, GROWTH_CAP)
    if g <= 0:
```

**Verify:** `g = min(g, GROWTH_CAP)` appears once in `process_ticker()`, after the `return row` early exit and before the `if g <= 0` floor check.
